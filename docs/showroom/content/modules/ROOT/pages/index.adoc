= Aegis -- Zero-Trust Identity for Agentic AI
:navtitle: Welcome

[.lead]
An AI agent that can *act on real systems* -- but holds *no stored credential*,
*reads as the human* it works for, and can only *change* anything after a
*human approves it in a console*. Every grant is short-lived and self-expiring;
every action is audited.

== Why it matters

* *Nothing to steal.* The agent carries only a cryptographic identity (SPIFFE), not an API key. A prompt-injection or pod compromise yields no usable credential.
* *Acts as the human.* Reads are delegated -- the downstream tool sees the *person*, so its own logs, RBAC, and rate limits apply.
* *No change without a human.* Privileged actions are blocked by default and require an explicit approval in a console -- by someone *other* than the requester.
* *No standing privilege.* Approval mints a scoped grant that does the one action and then expires by itself. Nobody has to remember to revoke it.

== Proven end to end

A credential-less agent did the real pfSense journey:
*read `200`* (as the user) -> *write `403`* (blocked) -> *human approves in the console* -> *write `200`* (a real firewall rule) -> *grant expires* -> *fully audited*. The same control plane works for any tool the agent reaches (Kubernetes, cloud, DNS, databases).

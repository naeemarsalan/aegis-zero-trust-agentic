= Platform overview
:navtitle: Platform overview

[.lead]
Aegis is four layers deep: an isolated agent sandbox, a shared identity and trust plane,
a policy and elevation control plane, and the real tools it governs.
One custom Go service wires them together; everything else is a supported product or established open source.

== Layer guide

* *Agent sandbox.* The agent holds only a SPIFFE SVID -- no API keys, no stored secrets.
  It runs in a native OpenShell sandbox on OpenShift (Kata / Confidential Containers = roadmap).
* *Identity & trust.* SPIRE issues workload SVIDs; Keycloak exchanges the agent's identity for the
  user's (RFC 8693); Vault stores per-agent consent grants and per-tool secrets fetched in-memory.
* *Policy & elevation.* The gateway (agentgateway) routes every MCP call through a two-step chain --
  Kyverno ext_authz decides allow/deny; `ext-proc-delegation` swaps the identity and injects the
  downstream credential.  Writes are blocked by default until the Approval Console issues a
  short-lived, scoped capability JWT (write = 5 min single-use; exec = 30 min).
* *Real tools.* Any MCP-speaking tool gets these guarantees with zero agent-side changes.
  The proven path is pfSense and Kubernetes; the pattern generalises to cloud, DNS, databases.
* *Cross-cutting.* ArgoCD app-of-apps is the GitOps source of truth.
  OTel feeds Loki/Grafana with tamper-evident (WORM) structured events; tool arguments are
  SHA-256 hashed before logging.

== Key design choices

[cols="1,2",options="header"]
|===
| Choice | Why

| One custom component (`ext-proc-delegation`)
| Single audit surface; one binary to review, maintain, and replace

| Fail closed everywhere
| Both the authz filter and the mutation filter are required; any error returns a denial

| Structural auto-revoke
| Capability JWTs are short-lived by construction -- no cron, no reconciler, no human in the revoke path

| Approver ≠ requester, enforced at the mint layer
| Four-eyes is a property of the system, not a convention the approver can override
|===
